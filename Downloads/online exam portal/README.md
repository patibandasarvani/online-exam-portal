# Online Exam Portal

A modern, responsive web application for conducting online examinations with ease and efficiency.

## Features

- **Student Login**: Secure authentication for students to access exams
- **Admin Login**: Administrative interface for exam management
- **Multiple Exam Categories**: Support for various subjects (Mathematics, Science, Computer Science)
- **Timer Functionality**: Automatic exam submission when time expires
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, intuitive interface with smooth animations
- **Notification System**: Real-time feedback for user actions

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The application will load automatically

### Usage

#### For Students:

1. Click on "Student Login" button
2. Enter your credentials (demo: any username/password)
3. Select an exam from the available list
4. Start the exam and complete within the time limit

#### For Administrators:

1. Click on "Admin Login" button
2. Enter admin credentials (demo: username: `admin`, password: `admin123`)
3. Access the admin dashboard (placeholder for future implementation)
